How It Works (Symmetric Encryption)

üîë Step 1: Shared Secret Key

Alice and Bob must somehow exchange a shared secret key beforehand. This could be done securely using RSA key exchange or Diffie-Hellman. Once they have the same key, they can encrypt/decrypt messages.

‚úâÔ∏è Step 2: Encrypt & Send

To send a message, Alice encrypts it using AES-256 with the shared key. The same key is used for both encryption and decryption. AES is much faster than RSA for bulk data!

üîì Step 3: Decrypt Received

Bob receives the encrypted message and decrypts it using the same shared key. Without this key, Eve cannot decrypt anything, even though she can intercept all messages.
